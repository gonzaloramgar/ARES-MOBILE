⚡ ARES Design Preview

Necesitas instalar Node.js para continuar

⚠️ Node.js no detectado
El servidor ARES Design Preview requiere Node.js para funcionar. Sigue los pasos abajo para instalarlo.

Instalación Paso a Paso

1
Descargar Node.js
Haz clic en el botón de abajo para descargar Node.js versión LTS (Long Term Support).
↓ Descargar Node.js LTS
2
Ejecutar el Instalador
Una vez descargado, abre el archivo y sigue el asistente de instalación.
Busca en Descargas: node-vXX.XX.XX-x64.msi
3
Configuración IMPORTANTE ⭐
Durante la instalación, asegúrate de que estas opciones estén marcadas:
☑ Add to PATH (MUY IMPORTANTE)
☑ Automatically install the necessary tools...
⚠️ Si NO marcas "Add to PATH", no funcionará. Vuelve a correr el instalador y marca esta opción.
4
Reiniciar la Computadora
Después de completar la instalación, REINICIA la computadora para que el PATH se actualice.
Este paso es crítico. Sin reiniciar, Windows no reconocerá Node.js.
5
Ejecutar el Servidor de Nuevo
Después de reiniciar, vuelve a hacer doble clic en start-server.bat
C:\Users\grg30\Desktop\PROYECTOS\ProyectoARES\design-preview-server\start-server.bat
6
Abre el Navegador
Si todo va bien, verás este mensaje en la terminal:
[OK] Iniciando servidor...
[*] Abre en el navegador: http://localhost:3000
→ Ir a http://localhost:3000

Verificación Rápida

Si quieres verificar que Node.js está instalado correctamente, abre PowerShell o CMD y corre:

node --version
npm --version

Deberías ver números de versión (ej: v20.10.0)
