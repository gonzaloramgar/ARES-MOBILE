# ARES Mobile - Setup Instructions

## First-time project setup

The Gradle wrapper scripts (`gradlew`, `gradlew.bat`) and the binary `gradle/wrapper/gradle-wrapper.jar`
cannot be generated by file write operations. You must generate them using one of these methods:

### Option A: Android Studio (recommended)
1. Open Android Studio
2. Select **Open** and navigate to this directory (`ARES-mobile/`)
3. Android Studio will detect the project and offer to generate the Gradle wrapper automatically
4. Accept and let it sync

### Option B: Command line (requires Gradle installed globally)
From within the `ARES-mobile/` directory, run:
```
gradle wrapper --gradle-version 8.10.2
```

### Option C: Copy from another Android project
Copy `gradlew`, `gradlew.bat`, and `gradle/wrapper/gradle-wrapper.jar` from any existing Android project.

## After generating the wrapper
```bash
# On macOS/Linux
./gradlew assembleDebug

# On Windows
gradlew.bat assembleDebug
```
